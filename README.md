# Visualización crítica del paisaje cañero — Valle del Cauca

Este repositorio contiene el código fuente de una **plataforma web interactiva** desarrollada con React, orientada a la visualización crítica del paisaje cañero en el Valle del Cauca. El proyecto articula investigación académica, análisis visual y diseño editorial digital para problematizar las narrativas hegemónicas de desarrollo asociadas a la agroindustria de la caña de azúcar.

El sitio combina texto analítico, archivo fotográfico, viñetas narrativas y galerías visuales para cuestionar la naturalización del monocultivo como sinónimo de progreso, eficiencia y modernidad.

---

## Enfoque conceptual

El proyecto se inscribe en los estudios críticos del desarrollo, el paisaje y la visualidad. Parte de la premisa de que el paisaje no es un fondo neutro ni exclusivamente natural, sino una **construcción histórica, política y estética**.

Ejes conceptuales abordados:
- Narrativas del desarrollo y regímenes de verdad
- Blanqueamiento del paisaje
- Monocultivo, extractivismo y ordenamiento territorial
- Paisaje cañero como dispositivo simbólico
- Archivo visual y contra-imagen como formas críticas de lectura territorial

---

## Estructura del sitio

La aplicación está organizada en las siguientes secciones principales:

- **Inicio**  
  Presentación general del proyecto y su marco conceptual.

- **Metodología**  
  Describe el proceso de investigación, análisis visual y construcción narrativa.  
  Incluye:
  - Ejes analíticos interactivos
  - Artículos desplegables
  - Figuras enlazadas con referencias cruzadas

- **Viñetas**  
  Textos narrativos que exploran el paisaje desde experiencias situadas y aproximaciones sensibles.

- **Contra-imagen**  
  Galería visual que tensiona las imágenes institucionales y oficiales del paisaje cañero.

- **Archivos**  
  Dashboard visual con más de 130 imágenes, cada una con su referencia completa.  
  Permite navegación entre imágenes, referencias y citas desde el texto principal.

---

## Archivo visual y referencias

Las imágenes provienen de archivos históricos, publicaciones editoriales, bibliotecas digitales y registros contemporáneos.  
Todas las imágenes están referenciadas de forma explícita y se enlazan desde:
- El texto analítico
- La sección de archivos visuales

El manejo de assets estáticos se realiza desde el directorio `public/` para garantizar compatibilidad en producción (Netlify).

---

## Tecnologías utilizadas

- **React** (con Vite)
- **React Router** para navegación SPA
- **CSS personalizado** (sin frameworks UI)
- Gestión de assets estáticos desde `public/assets`
- Deploy en **Netlify**

Características técnicas destacadas:
- Renderizado dinámico de galerías
- Modal interactivo con navegación por teclado
- Búsqueda y filtrado de archivos visuales
- Enlaces cruzados entre texto, figuras y referencias
- Soporte para rutas directas (SPA)

---

## Objetivo del proyecto

El objetivo principal no es solo informar, sino **desnaturalizar el paisaje cañero**, evidenciando sus dimensiones históricas, políticas y simbólicas. El proyecto funciona como:
- Herramienta de análisis crítico
- Archivo visual comentado
- Plataforma de investigación–creación

---

## Estado del proyecto

El proyecto se encuentra funcional y desplegado. Está diseñado para ser escalable, permitiendo:
- Añadir nuevas viñetas
- Incorporar más material de archivo
- Expandir los ejes analíticos
- Integrar otros formatos visuales o narrativos

# Critical Visualization of the Sugarcane Landscape — Valle del Cauca

This repository contains the source code for an **interactive web platform** built with React, focused on the critical visualization of the sugarcane landscape in Valle del Cauca, Colombia. The project integrates academic research, visual analysis, and digital editorial design to question dominant development narratives tied to the sugarcane agroindustry.

The site combines analytical text, photographic archives, narrative vignettes, and visual galleries to challenge the naturalization of monoculture as a symbol of progress, efficiency, and modernity.

---

## Conceptual framework

The project is grounded in critical development studies, landscape theory, and visual studies. It understands landscape not as a neutral or purely natural backdrop, but as a **historical, political, and aesthetic construction**.

Core thematic axes include:
- Development narratives and regimes of truth
- Landscape whitening
- Monoculture, extractivism, and territorial ordering
- Sugarcane landscape as a symbolic device
- Visual archives and counter-image as critical tools

---

## Site structure

The application is organized into the following main sections:

- **Home**  
  General introduction to the project and its conceptual scope.

- **Methodology**  
  Details the research process, visual analysis, and narrative construction.  
  Includes:
  - Interactive analytical axes
  - Expandable articles
  - Figures linked to references

- **Vignettes**  
  Narrative texts that approach the landscape from situated and experiential perspectives.

- **Counter-image**  
  A visual gallery that questions institutional and official representations of the sugarcane landscape.

- **Archives**  
  A visual dashboard containing over 130 images, each with full bibliographic references.  
  Enables navigation between images, references, and citations from the main text.

---

## Visual archive and references

Images are sourced from historical archives, editorial publications, digital libraries, and contemporary records.  
All images are explicitly referenced and cross-linked from:
- Analytical text sections
- The visual archive dashboard

Static assets are managed through the `public/` directory to ensure correct behavior in production environments (Netlify).

---

## Technologies used

- **React** (with Vite)
- **React Router** for SPA navigation
- **Custom CSS** (no UI frameworks)
- Static asset handling via `public/assets`
- Deployment on **Netlify**

Key technical features:
- Dynamic gallery rendering
- Interactive modal with keyboard navigation
- Search and filtering for visual assets
- Cross-linked text, figures, and references
- Full SPA routing support

---

## Project goal

Rather than merely informing, the project aims to **denaturalize the sugarcane landscape**, revealing its historical, political, and symbolic dimensions. It operates as:
- A critical analysis tool
- A curated visual archive
- A research–creation platform

---

## Project status

The platform is fully functional and deployed. It is designed to be scalable, allowing for:
- New narrative vignettes
- Additional archival material
- Expansion of analytical axes
- Integration of new visual or narrative formats
